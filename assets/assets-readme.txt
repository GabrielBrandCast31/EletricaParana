DIRETÓRIO DE ASSETS - ELÉTRICA PARANÁ LANDING PAGE

Coloque neste diretório os arquivos de mídia reais para a Landing Page antes de publicar:

1. favicon-placeholder.png -> Substitua pelo favicon real da empresa (16x16 ou 32x32 pixels, formato .png ou .ico).
2. og-image-placeholder.jpg -> Substitua pela imagem de compartilhamento em redes sociais (recomendado: 1200x630 pixels, formato .jpg).
3. Logo e fotos reais:
   - A página utiliza logotipos em formato SVG inline de alta performance para carregamento instantâneo (LCP < 2,5s). Caso prefira usar arquivos externos de imagem (como logo.png), altere as tags .logo-svg em index.html.
   - Substitua as fotos dos placeholders na seção de prova social pelas imagens reais das lojas de Nova Mutum, Sinop, Couto Magalhães e Matriz FEB.
